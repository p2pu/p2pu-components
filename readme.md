## Development

### To run demo
```
npm i
npm run start
```
The demo runs on `localhost:3001`

### To build the demo

```
npm run build:demo
```

### To build the main package

```
npm i
npm run build:main
```

### To publish a new version

```
npm adduser
npm version patch
npm publish
```

You'll need the right credentials to publish to p2pu-search-cards.
